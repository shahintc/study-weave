ğŸ“ Description
Study Weave is a web application designed for the CS 319 Object-Oriented Software Engineering course. It provides a platform for researchers to conduct systematic human-subject studies comparing software engineering artifacts (source code, test cases, UML diagrams) generated by both humans and AI (LLMs).

The system uses React.js for a dynamic frontend and Node.js for the backend, demonstrating practical skills in OOD, full-stack development, and collaborative project management.

1. Effortless Study Management

A side-by-side comparison interface with synchronized scrolling and a "blinded mode" ensures unbiased, high-quality feedback.

3. Actionable Analytics
Real-time dashboards visualize study progress and participant completion rates, with full data export capabilities for deeper research.
